üåü Exercise 2: HTML Form with POST Method

üìù Instructions:

Fill out the form below and click "Send". The form uses the POST method.

Question: Where will the sent data appear? Hint: Look at the Network Tab

‚úÖ Answer: Where will the sent data appear?

When you submit this form with POST method, the data will NOT appear in the URL. Instead, it will be sent in the HTTP request body.

üîç How to see the POST data:

  1. Open your browser's Developer Tools (F12)
  2. Go to the Network tab
  3. Fill out and submit the form above
  4. Look for the request to this page
  5. Click on that request
  6. Go to the Request or Payload section
  7. You'll see the form data there in the request body

Note: Since this is a static HTML file, the form won't actually process the data, but you can still see it being sent in the Network tab!

‚úÖ Answer: What happens to nested objects?

                
When converting JavaScript objects to JSON:

                
    
                    
  • Nested objects are preserved - They remain as nested structures in the JSON
    • 
                    
  • Property names are automatically quoted with double quotes
    • 
                    
  • Values are converted to appropriate JSON data types (strings, numbers, booleans, arrays, objects)
    • 
                    
  • Functions are omitted (JSON doesn't support functions)
    • 
                    
  • Undefined values are omitted
    • 
                

                
In our Mario game example, the characters object contains nested objects for each character (mario, bowser, princessPeach), and these nested structures are maintained in the JSON output.
